# Contributing

Thank you for your interest in this repository.

This project is maintained solely by Nexato GmbH as a reference implementation for customers using nexcore.

At this time we are **not accepting**:

- Pull Requests
- feature implementations
- code contributions

GitHub Issues are not used as a support channel.

If you have questions regarding nexcore, please use the official Nexato support channels.

If you believe you have identified an issue within the example implementation itself, you are welcome to open an Issue describing the behavior. While we appreciate the feedback, we cannot guarantee a response or implementation.
