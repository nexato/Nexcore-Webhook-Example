# Security Policy

## Reporting Security Vulnerabilities

If you discover a security vulnerability affecting nexcore or one of the examples contained in this repository, please do **not** report it publicly through GitHub Issues.

Instead, please contact Nexato directly through the official support channels and provide as much information as possible to allow us to investigate the issue.

Please include, where applicable:

- a description of the vulnerability
- steps to reproduce
- affected versions
- proof of concept (if available)
- potential impact

We appreciate responsible disclosure and will investigate all legitimate reports as quickly as possible.

---

## Supported Versions

This repository contains example code only.

No guarantees are made regarding long-term maintenance or compatibility with future nexcore releases.

The examples may be updated, modified or discontinued at any time.
